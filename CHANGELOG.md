# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-12-26

### Added

- Initial release of Vento Image gallery library
- TypeScript-based state management with reducer pattern
- Responsive image gallery with CSS-based rendering
- Plugin system for extensible functionality
- Core plugins:
  - AutoplayPlugin for automatic slideshow
  - KeyboardPlugin for keyboard navigation
  - TouchPlugin for touch/swipe gestures
  - FullscreenPlugin for fullscreen mode
  - VideoPlugin for YouTube/Vimeo video support
- Multiple transition types: slide, fade, crossfade
- Navigation options: thumbnails, dots
- Mobile-friendly touch and swipe support
- Accessibility features with ARIA attributes
- Comprehensive TypeScript definitions
- Build system with Vite
- Test suite with Vitest
